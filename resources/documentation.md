# Dokumentasi Aplikasi Inventaris Studio Musik
Aplikasi desktop berbasis **Java Swing** untuk mengelola peralatan dan perlengkapan studio musik.
Aplikasi ini dirancang sebagai sistem **CRUD** (Create, Read, Update, Delete) dan mengikuti arsitektur **MVC (Model–View–Controller)** yang bersih.
## Fitur Utama
- Menambahkan, Melihat, Memperbarui, Menghapus (CRUD) data per
alatan
- Antarmuka GUI desktop menggunakan Java Swing
- Validasi input dengan dialog peringatan
- Arsitektur bersih menggunakan MVC (Model–View–Controller)
- Antarmuka pengguna yang sederhana dan bersih
## Teknologi yang Digunakan
- Java JDK 8+
- Java Swing
- Maven (alat build)
- VS Code
- Git (kontrol versi)
## Arsitektur
Proyek ini menggunakan arsitektur **MVC (Model–View–Controller)**:
### Model
Bertanggung jawab untuk representasi data.
- `Equipment.java`
### View
Menangani semua komponen GUI (Java Swing).
- `MainFrame.java`
- `FormPanel.java`
- `TablePanel.java`
### Controller
Menangani logika aplikasi dan operasi CRUD.
- `EquipmentController.java`
Pemecahan ini meningkatkan:
- Keterbacaan kode
- Pemeliharaan
- Skalabilitas
